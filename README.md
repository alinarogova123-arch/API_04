# Космический Телеграм
## fetch_nasa_apods.py

Скрипт скачивает фотографии дня NASA (apod) в папку "images", если папки с таким названием нет в директории скрипта, она появится.

### Как запустить

Потребуется API ключ NASA, получить его можно на сайте https://api.nasa.gov, затем
скопировать севрисный ключ и вставить в файл `.env`, присвоив переменной окружения `NASA_API_KEY`, как в примере ниже:

`NASA_API_KEY=jf85heik58fydjdkfifundk7`

Скрипт запускается командой:

```bash
python fetch_nasa_apods.py
```

## fetch_spacex_launches.py

Скрипт скачивает фотографии запусков SpaceX в папку "images", если папки с таким названием нет в директории скрипта, она появится.

### Как запустить

Скрипт запускается командой:

```bash
python fetch_spacex_launches.py
```

Командой выше скрипт скачает фотографии последнего запуска, для загрузки фотографий других запусков нужно указать номер запуска в качестве аргумента как в примере ниже:

```bash
python fetch_spacex_launches.py 63
```
