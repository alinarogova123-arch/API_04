# Космический Телеграмм

Автоматизирует загрузку фотографий о космосе на ваше хранилище и постинг их в телеграмм канале.

## fetch_nasa_apods.py

Скрипт скачивает фотографии дня NASA (apod) в папку "images", если папки с таким названием нет в директории скрипта, она появится.

### Как запустить

Потребуется API ключ NASA, получить его можно на сайте https://api.nasa.gov, затем
скопировать ключ и вставить в файл `.env`, присвоив переменной окружения `NASA_API_KEY`, как в примере ниже:

`NASA_API_KEY=jf85heik58fydjdkfifundk7`

Скрипт запускается командой:

```bash
python fetch_nasa_apods.py
```

## fetch_spacex_launches.py

Скрипт скачивает фотографии запусков SpaceX в папку "images", если папки с таким названием нет в директории скрипта, она появится.

### Как запустить

Скрипт запускается командой:

```bash
python fetch_spacex_launches.py
```

Командой выше скрипт скачает фотографии последнего запуска, для загрузки фотографий других запусков нужно указать номер запуска в качестве аргумента как в примере ниже:

```bash
python fetch_spacex_launches.py 63
```

## posting_telegram_bot.py

Скрипт безконечно постит скачанные фотографии в телеграм канал, пока запущен. Сначала постит по порядку все фотографии, затем пермешивает их, снова постит, снова пермешивает, снова постит...

### Как запустить

Вам потребуется API token вашего бота телеграм, а так же chat id вашего канала, так же нужно сделать вашего бота администраторм вашего канала.
API token вашего бота присвоить переменной `POSTING_TELEGRAM_BOT_API_KEY`, chat id вашего канала присвоить переменной `TLGRM_CHAT_ID` в файле .env как в примере ниже:

``` bash
POSTING_TELEGRAM_BOT_API_KEY=jf85heik58fydjdkfifundk7
TLGRM_CHAT_ID=@channel_id
```

По умолчанию интервал между постами составляет 4 часа, скрипт запускается командой:

```bash
posting_telegram_bot.py
```

Для выбора интервала между постами необходимо указать время в часах в качестве аргумента, как в примере ниже:

```bash
posting_telegram_bot.py 2
```

## Необходимое окружение

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```
